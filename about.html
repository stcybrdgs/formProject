About my JSP Form Project


This project demonstrates how the JavaServer Pages (JSP) technology may be used to perform server-side processing on a simple HTML Form.

To go to the demo, click JSP Form Demo in the above menu (or click here).


One advantage in using JSP for web applications is that it provides for server-side functionality as well as Object Oriented Programming (OOP) techniques. Such flexibility allows web developers who enjoy Java to cover a lot of bases with just one language.

This project is being staged from OpenShift Online, which is a Platform as a Service (PaaS) provided by Red Hat. It allows developers to focus their time and energy on building applications without having to fret over the server administration details.

This project has no database on the back-end, so Javascript was used to generate the drop menus and the confirmation key.

The code for this project is posted in the menu area to your left. Feel free to view it in your browser or to print it at your discretion.
